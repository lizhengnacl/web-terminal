# 007-在线命令执行器

## 产品概述

一个现代Web风格的在线命令执行器，允许用户通过浏览器访问服务器终端，执行命令、管理文件，提供接近本地终端的使用体验。

---

## 目标用户

**主要用户**：开发者、运维工程师

**使用场景**：
- 个人开发调试 - 在远程服务器上快速执行命令
- 服务器运维管理 - 轻量级的服务器管理工具
- 团队协作 - 共享服务器访问（初期暂不做复杂权限）

---

## 核心功能

### 1. 终端交互
- **实时流式输出**：命令输出像本地终端一样实时显示
- **命令输入**：支持多行命令输入
- **命令历史**：记录执行过的命令，支持上下键切换
- **命令历史搜索**：快速搜索和复用历史命令
- **自动补全**：Tab键自动补全命令和路径

### 2. 多会话管理
- **多个终端标签页**：同时打开多个独立的终端会话
- **标签页管理**：新建、关闭、切换标签页
- **会话保持**：刷新页面试图恢复会话（尽力而为）

### 3. 文件管理
- **文件上传**：通过Web界面上传文件到服务器
- **文件下载**：从服务器下载文件到本地
- **目录浏览**：（可选）可视化目录树展示

### 4. 快捷操作
- **常用命令快捷按钮**：一键执行 ls、pwd、cd ~ 等常用命令
- **清空终端**：一键清空当前终端输出
- **复制输出**：一键复制终端输出内容

### 5. 历史记录
- **完整审计日志**：记录谁在什么时候执行了什么命令
- **输出保存**：保存命令执行的完整输出结果
- **历史查看**：查看历史命令执行记录

### 6. 安全认证
- **密码认证**：通过密码登录后才能使用
- **会话管理**：登录状态保持，超时自动登出

---

## 技术栈

- **后端**：Node.js + Express
- **实时通信**：Socket.io（用于流式输出）
- **终端模拟**：node-pty（伪终端）
- **前端框架**：React 18 + TypeScript
- **UI组件**：可选择 Ant Design / Material-UI / 自定义
- **构建工具**：Vite
- **样式**：Tailwind CSS / CSS Modules

---

## 用户流程

```
访问页面 → 密码登录 → 打开终端 → 执行命令 → 查看输出
```

### 首次使用
1. 打开页面显示登录界面
2. 输入密码验证身份
3. 登录成功后显示默认终端标签页
4. 开始执行命令

### 日常使用
1. 打开页面自动恢复登录状态（如未过期）
2. 直接在终端中输入命令
3. 使用标签页管理多个会话
4. 查看历史命令记录

## 界面设计

### 整体布局
```
┌───────────────────────────────────────────────────────────────┐
│  [Logo]  终端  文件  历史  [设置]              [用户名] [退出]  │  ← 顶部导航栏
├──────────────┬─────────────────────────────────────────────────┤
│              │  [+] 终端1  [×] 终端2  [×] 终端3               │  ← 标签栏
│  快捷按钮    ├─────────────────────────────────────────────────┤
│  • ls        │                                                 │
│  • pwd       │              终端输出区域                       │
│  • cd ~      │            (实时流式显示)                       │
│  • clear     │                                                 │
│              │                                                 │
│              ├─────────────────────────────────────────────────┤
│              │  > 命令输入框 (支持自动补全)                    │
└──────────────┴─────────────────────────────────────────────────┘
```

### 侧边栏快捷按钮
- ls - 列出当前目录
- pwd - 显示当前路径
- cd ~ - 回到用户主目录
- clear - 清空终端
- 上传文件
- 下载文件

### 文件管理页面
- 目录树展示
- 文件列表
- 上传/下载操作

### 历史记录页面
- 时间线展示命令执行历史
- 筛选和搜索
- 查看详细输出

## 边缘场景考虑

1. **长耗时命令**：支持后台执行，不阻塞界面
2. **大量输出**：自动滚动，支持暂停自动滚动查看历史
3. **网络中断**：断线重连机制，尽力恢复会话
4. **命令执行失败**：清晰显示错误信息和退出码
5. **特殊字符处理**：正确处理ANSI颜色码等终端控制字符
6. **并发访问**：多人同时使用时的会话隔离
7. **安全性**：防止命令注入攻击（虽然是自用工具，但基本防护要有）
8. **资源限制**：防止执行消耗过多资源的命令
