# TC-025: 浏览目录树

**模块**：文件管理页面
**测试类型**：功能测试
**优先级**：P1

## 前置条件
1. 用户已成功登录系统
2. 已进入文件管理页面
3. 服务器上存在目录结构：
   - /home/user/
     - documents/
       - file1.txt
     - downloads/
       - file2.pdf

## 测试步骤
1. 在目录树中点击 "/home/user/documents" 目录节点
2. 观察目录树变化
3. 点击 "/home/user/documents" 目录节点再次折叠

## 预期结果
1. 点击目录节点后，展开显示子目录和文件
2. 显示 "file1.txt" 文件节点
3. 目录节点图标从关闭状态变为打开状态
4. 再次点击目录节点，折叠子目录和文件
5. 目录节点图标恢复关闭状态
