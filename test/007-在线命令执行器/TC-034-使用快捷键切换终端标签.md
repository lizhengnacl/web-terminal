# TC-034: 使用快捷键切换终端标签

**模块**：全局功能
**测试类型**：功能测试
**优先级**：P1

## 前置条件
1. 用户已成功登录系统
2. 当前在终端页面
3. 当前有3个终端标签："终端1"、"终端2"、"终端3"
4. 当前活跃标签为"终端1"

## 测试步骤
1. 按下快捷键 Ctrl/Cmd + 2
2. 按下快捷键 Ctrl/Cmd + Tab
3. 按下快捷键 Ctrl/Cmd + Shift + Tab

## 预期结果
1. 按下 Ctrl/Cmd + 2 后，切换到"终端2"标签
2. "终端2"标签显示活跃态
3. 按下 Ctrl/Cmd + Tab 后，切换到下一个标签"终端3"
4. "终端3"标签显示活跃态
5. 按下 Ctrl/Cmd + Shift + Tab 后，切换到上一个标签"终端2"
6. "终端2"标签显示活跃态
