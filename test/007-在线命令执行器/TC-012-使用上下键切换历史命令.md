# TC-012: 使用上下键切换历史命令

**模块**：终端页面
**测试类型**：功能测试
**优先级**：P1

## 前置条件
1. 用户已成功登录系统
2. 已进入终端页面
3. 在当前终端中已执行过以下命令：
   - "ls -la"
   - "pwd"
   - "cd ~"

## 测试步骤
1. 点击命令输入框，确保输入框有焦点
2. 按下键盘 ↑ 键
3. 再次按下键盘 ↑ 键
4. 按下键盘 ↓ 键

## 预期结果
1. 第一次按 ↑ 键：命令输入框显示最近执行的命令 "cd ~"
2. 第二次按 ↑ 键：命令输入框显示上一条命令 "pwd"
3. 按 ↓ 键：命令输入框显示下一条命令 "cd ~"
4. 可继续按 ↑ 或 ↓ 键浏览历史命令
5. 按 Enter 键可执行当前显示的命令
