# TC-020: 大量输出处理

**模块**：终端页面
**测试类型**：性能测试
**优先级**：P1

## 前置条件
1. 用户已成功登录系统
2. 已进入终端页面

## 测试步骤
1. 在命令输入区输入命令 "cat /var/log/largefile.log"（假设该文件有超过10000行）
2. 按下键盘 Enter 键执行命令
3. 观察终端输出区

## 预期结果
1. 命令开始执行，输出实时追加到终端输出区
2. 当输出超过10000行时，自动截断早期输出
3. 保留最新的输出内容
4. 显示提示："输出已截断，仅显示最新内容"
5. 终端性能保持流畅，不出现卡顿
6. 滚动操作响应正常
