# TC-016: 网络中断重连

**模块**：全局功能
**测试类型**：异常测试
**优先级**：P0

## 前置条件
1. 用户已成功登录系统
2. 已进入终端页面
3. WebSocket连接正常

## 测试步骤
1. 断开网络连接（关闭WiFi或拔掉网线）
2. 观察系统反应
3. 恢复网络连接

## 预期结果
1. 网络断开时，显示断线提示："网络连接已断开，正在尝试重连..."
2. 命令输入框禁用，无法输入新命令
3. 系统自动尝试重连WebSocket
4. 网络恢复后，WebSocket自动重新连接
5. 连接成功后，提示"网络已恢复"
6. 命令输入框恢复可用状态
7. 之前的终端会话状态保持
