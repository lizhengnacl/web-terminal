# TC-027: 搜索历史命令

**模块**：历史记录页面
**测试类型**：功能测试
**优先级**：P0

## 前置条件
1. 用户已成功登录系统
2. 已执行过多个命令：
   - "ls -la"
   - "pwd"
   - "cat file.txt"
   - "grep pattern file.txt"
3. 已进入历史记录页面

## 测试步骤
1. 在搜索框中输入关键词 "file"

## 预期结果
1. 历史列表实时过滤
2. 只显示包含关键词 "file" 的命令：
   - "cat file.txt"
   - "grep pattern file.txt"
3. 其他命令被隐藏
4. 搜索框显示输入的关键词
